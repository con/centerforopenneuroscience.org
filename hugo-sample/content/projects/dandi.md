Distributed Archives for Neurophysiology Data Integration (DANDI) is a platform for publishing, sharing, and processing neurophysiology data funded by the BRAIN Initiative. DANDI allows researchers to:

- Upload and share neurophysiology data in standardized formats
- Discover and access published datasets
- Organize and validate data using the Neurodata Without Borders (NWB) standard
- Process data using cloud-based computational resources

The platform serves as a critical infrastructure component for the neuroscience community, enabling reproducible research and data sharing.
