[tinuous](https://github.com/con/tinuous/) is a command for downloading build logs and (for GitHub only) artifacts and release assets for a GitHub repository from GitHub Actions, Travis-CI.com, and/or Appveyor. By downloading them all, and optionally placing them under [DataLad](#datalad) control you can establish the backup, distribution, and convenient harmonious access to all those artifacts.
