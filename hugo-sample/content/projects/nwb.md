[NWB:N](https://www.nwb.org/) is a data standard for neurophysiology, providing neuroscientists with a common standard to share, archive, use, and build analysis tools for neurophysiology data. It is a standard supported by [BIDS](bids) and the [DANDI archive](dandi).
