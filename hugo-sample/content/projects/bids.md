The Brain Imaging Data Structure (BIDS) is a standard for organizing and describing neuroimaging and behavioral data. BIDS was developed by the neuroimaging community to improve reproducibility and facilitate data sharing.

Key features:
- Standardized directory structure and file naming conventions
- Machine-readable metadata descriptions
- Support for multiple neuroimaging modalities (MRI, EEG, MEG, iEEG, PET)
- Growing ecosystem of BIDS-compatible analysis tools
- Community-driven development and governance

CON members actively participate in BIDS development, serving on steering committees, contributing to specification development, and creating BIDS-compatible tools and datasets.
