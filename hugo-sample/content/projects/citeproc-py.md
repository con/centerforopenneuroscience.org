citeproc-py is a Python implementation of a processor for the Citation Style Language (CSL), a format for describing citation and bibliography formatting rules. This project was adopted by CON to ensure its continued maintenance and development for the scientific community.

Key features:
- Full CSL 1.0.1 compatibility
- Support for all major citation styles
- Extensible architecture for custom formatting
- Integration with reference management tools
- Active maintenance and community support

CON adopted this project to ensure reliable citation processing capabilities remain available for the broader scientific software ecosystem.
